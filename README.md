dusnet是一个高性能、轻量级、高扩展性的网络通信框架，适用于多重设备节点间的互联互通，目前已成功应用的领域有智慧农业、智能家居、公交车、铁路干线系统、监控运维系统等。

dusnet主要拥有以下特性：

- 性能优秀，单机百万级消息并发能力
- 可兼容多种自定义规范协议，快速生成协议结构体、编解码组件
- 支持多种部署模式，兼容不同网络场景（单机部署、普通集群、主从集群、容器化、跳板机）
- 内置消息负载均衡插件，业务层级的负载均衡能满足一些额外的通信需求
- 每个节点根据网络链路都可作注册中心，亦可接入第三方注册中心来达到服务治理的目的，已支持（etcd、zookeeper）
